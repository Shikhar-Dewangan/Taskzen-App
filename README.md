# 🚀 TaskZen - Modern Task Manager

A modern and responsive task management web application built with **React**, **Redux Toolkit**, **Tailwind CSS**, and **Appwrite**.

TaskZen helps users efficiently manage daily tasks with a clean UI, real-time updates, and a powerful backend.

---

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- ➕ Add new tasks
- ✏️ Edit tasks inline
- ❌ Delete tasks
- ✅ Toggle task completion
- 📊 Dashboard with:
  - Total tasks
  - Completed tasks
  - Pending tasks
- 🕒 Recent Tasks section with smart date formatting
- 📱 Fully responsive (Mobile + Desktop)
- ⚡ Real-time UI updates using Redux Toolkit

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS

### State Management
- Redux Toolkit

### Backend
- Appwrite (Authentication + Database)
